# 📊 SalesPulse

An enterprise-grade, lightning-fast Sales Dashboard and Business Intelligence (BI) Platform tailored for operations in Bangladesh (**BDT**). Powered by React, Vite, Tailwind CSS, and optimized for both high-density desktop monitors and ultra-compact mobile layouts.

---

## 🚀 Core Capabilities & Feature Roadmap

### 🏁 1. Pipeline & Targets Analytics
* **Sales Representative Target Achievements**: Live target-quota versus gross sales tracker across sales representatives with performance ranking and deficit indicators.
* **Instant Excel Exporting**: Standardized Excel/CSV formatted tabular exports directly from the data grid for offline reporting and review.
* **Custom Configurator Panels**: Interactive control systems to seamlessly adjust row limits, select specific buyer ranks, and toggle table columns to optimize workspace utilization.

### 💼 2. Customer & Buyer Intelligence
* **Unified Contact & Intelligence Deck**: Deep buyer profile auditing, outstanding debt tracing, and collection-to-invoice ratio breakdowns.
* **Granular Data View Controls**: Real-time filters and custom column toggles to instantly change visible metrics (e.g., Velocity, Contribution %, Collections).

### 📈 3. Dynamic Interactive Data Visualizations
* **Intelligent Global Slicers**: Centralized filter system across categories, accounts, branches, and date ranges.
* **High-Contrast Vector Charts**: Responsive area, bar, and trend charts visualizing revenue momentum, category health, and collection aging lifecycles.

### ⚙️ 4. Premium System Preferences & Settings
* **Beautiful Dashboard Controls**: Stylized tabbed controls for configuring targets/allocations, global system variables, policy audit levels, and company groups.
* **Tailored Aesthetics**: Interactive tabs decorated with rich contextual indicators (🎯, ⚙️, ⚖️, 🏢) utilizing micro-interaction spring effects.

---

## 📱 Mobile UI & Responsive Architecture
* **High-Density Navigation Deck**: Custom-crafted, bottom-aligned mobile application dock featuring low-friction, high-vibration layout nodes.
* **Grid-to-Flex Adaptive Collapsing**: Tables, charts, and configurations collapse natively into elegant vertical flow modules under mobile environments.

---

## 🛠️ Technology Stack & Development Pipeline

* **Frontend Engine**: React 18+ powered by Vite for instant Hot Module Replacement (HMR) bypasses and quick builds.
* **Styling Framework**: Modern Tailwind CSS utility design patterns using CSS variable themes.
* **Visual Data & Graphics**: Integrated standard `lucide-react` icons and custom SVG interactive representations.

### Setup and Local Execution

To start the local development suite, run:
```bash
npm install
npm run dev
```

To compile the application for production deployment:
```bash
npm run build
```
The compiled output will be generated inside `/dist` for easy drag-and-drop server uploads, Docker builds, or Cloud Run deployments.
